# recsys-learning-to-rank-catboost# RecSys Learning-to-Rank with CatBoost (MovieLens)

Небольшой проект для демонстрации оффлайн-оценки рекомендаций на логах взаимодействий user–item.
Собран пайплайн: генерация кандидатов, далее обучение моделей, далее сравнение по метрикам ранжирования.

dataset:  https://www.kaggle.com/datasets/grouplens/movielens-20m-dataset?select=rating.csv

## Что сделано
- Преобразование MovieLens ratings в implicit feedback (позитив: rating >= 4.0)
- Разбиение по времени на пользователя: последний позитив в тест, остальные в трейн
- Генерация кандидатов: 1 позитив + N негативов (negative sampling)
- Модели:
  - baseline по популярности
  - SGD логистическая регрессия на hashed признаках user/movie
  - sklearn boosting на числовых фичах
  - CatBoostClassifier (pointwise скоринг)
  - CatBoostRanker (YetiRank)
- Метрики: NDCG@K, Recall@K (HitRate@K), MRR@K, Precision@K
- Графики метрик и автоматический вывод лучшей модели по NDCG@10

## Фичи
- user_cnt, user_mean_rating
- item_cnt, item_mean_rating, item_pos_cnt
- признаки времени (hour, dow, month)
- жанры из movie.csv (если файл есть рядом)

## Как запустить
1. Положить `rating.csv` (и опционально `movie.csv`) в папку data.
2. Открыть ноутбук и выполнить ячейки сверху вниз.

Основные параметры (кол-во негативов, лимиты по пользователям и строкам) задаются в начале ноутбука.

## Замечания по ресурсам
CatBoost может требовать много памяти на больших выборках, поэтому в проекте есть ограничения по:
- числу пользователей в обучении/оценке
- числу негативов
- максимальному числу строк для обучения CatBoost

Это сделано, чтобы код стабильно запускался на обычном ноутбуке.

## Структура
- `notebooks/` — основной ноутбук с обучением и сравнением моделей
- `data/` — (опционально) файлы MovieLens
- `requirements.txt` — зависимости

## Результат
В конце ноутбука выводится таблица с метриками по моделям, строятся графики и печатается лучшая модель по NDCG@10.
